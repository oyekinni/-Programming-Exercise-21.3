# -Programming-Exercise-21.3
 Programming Exercise 21.3
This program counts how many Java keywords appear in a source file. It uses a set of keywords and checks each word in the file against that set. While reading the file, it ignores anything inside comments (both // and /* */) and also ignores anything inside strings. This is important because keywords inside comments or strings should not be counted. After filtering those out, the program splits the remaining code into words and counts how many are valid Java keyword
